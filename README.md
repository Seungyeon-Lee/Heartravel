# noname-shopping-mall

### Pakacages
서버 (가장 외부 폴더)와 프론트 (내부의 frontend 폴더)로 구성.

### How to run
```
npm install nodemon -g
npm install
cd frontend
npm install
```

이후 서버 폴더에서 nodemon start

프론트 폴더에서 npm run serve 하여 실행.

**호스팅 상에 올릴 때에는 npm run build 이후 node_modules를 포함하여 push**
